## Overview
This directory contains all the automated tests for the **Redistricting Map Contact Dashboard**. The tests cover:
- Asynchronous data fetching and rendering.
- UI state transitions (Theme toggling, search filtering).
- Modal interactions and conditional rendering based on contact statuses.

## Running Tests
Run the following command from the root of the project to execute the test suite:
```bash
npm run test
```
