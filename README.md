# Redistricting Map Contact Dashboard

A premium dashboard built to track outreach to government officials, media, and political organizations regarding the algorithm-based state voting districts project.

## Features
- **Modern UI**: Built with React, Vite, and Lucide icons.
- **Premium Aesthetics**: Features a sleek dark mode design with glassmorphism and subtle micro-animations.
- **Contact Tracking**: Easily update and monitor the status of your outreach for each contact.
- **Filtering**: Quickly search for specific individuals, roles, or organizations.
- **Categorization**: Contacts are grouped logically for easier navigation.

## Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

## Deployment

To deploy this project to GitHub Pages:
```bash
npm run deploy
```
